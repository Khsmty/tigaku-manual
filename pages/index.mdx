# はじめに

地学部へようこそ。

このマニュアルは、地学部で活動していくために必要な基礎知識や技術を紹介するものです。

地学部では、主に天体望遠鏡を使って、夜空にたくさん浮かぶ星を観察します。

なお、天体望遠鏡をはじめとする器類具は地学部、そして学校の貴重な財産です。器具が後輩たちの世代までずっと使えるように、丁寧に扱い、正しく片付けましょう。

また、器具を使うときは、安全に注意し、事故や怪我が起きないように気をつけてください。

ぜひ、このマニュアルを参考にして、器具の使い方や観測方法を自分の中へ吸収してください。

部員一人ひとりが協力しながら成長して、私たちでも見ることのできなかった難しい星を見られるようになることを願っています。
